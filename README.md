# Prueba técnica


## Instalación 

- Paso #1: Clonar repositorio 

*git clone https://github.com/Monky021/prueba-tecnica.git*
- Paso #2: Entrar a la carpeta del proyecto 

*cd prueba-tecnica*
- Paso #3: Instalar dependencias

*npm install*

- Paso #4: Ejecutar en local

*npm run dev*




## Versiones del aplicativo

- Node 18.12.1
- React 18.2.0
